第二章 变量和基本类型：
	
	1、c++的数据类型：
		基本数据类型：算数类型（字符、整型数、布尔值、浮点数）和空类型。
		非基本数据类型：数组、指针、引用、类、结构、联合、枚举。
	
	2、初始化和赋值的区别的：
		初始化不是赋值，初始化的含义是创建变量时赋予其一个初始值，而赋值的含义是把对象的当前值擦除，而以一个新值来替代。
	
	3、C++11的四种赋值语句
		int a = 0;
		int a = {0};		//列表初始化
		int a{0};			//列表初始化
		int a(0);
		如果我们使用列表初始化且初始值存在丢失信息的风险，则编译器将报错
		
	4、C++是一种静态类型的语言，其含义是在编译阶段检查类型。
	
	5、复合类型：指针、引用等
		1、引用：为对象起了另外一个名字，引用类型引用另一种类型
			通过将声明符写成&d的形式来定义引用类型，其中d是声明的变量名
		2、指针：是指向另一种类型的复合类型
			本身就是一个对象，允许对指针赋值拷贝、无需赋值
			int *p = 111	错误不能直接指向int型变量
			int *p = &val	初始化p为val的地址
			p = &ival		修改p指向ival的地址
			*p = 10			修改p指向的对象的值为10
